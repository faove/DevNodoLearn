# DevNodo Learn — Plataforma educativa de programación

## Objetivo
App web interactiva para enseñar programación a jóvenes +15 años.
Stack: React + Pyodide (Python en browser) + API de Claude como tutor AI.

## Fase 1 a implementar
- Lección 1: Variables y tipos en Python
- Editor de código ejecutable (Monaco Editor + Pyodide)
- Asistente Claude que responde dudas en contexto de la lección
- Ejercicios con validación automática

## Stack decidido
- React (Vite)
- Monaco Editor (editor de código)
- Pyodide (ejecutar Python en el browser sin backend)
- Anthropic API (tutor AI)
